# Patika Java101
```
Ask the user for their grades in Mathematics, Physics, Turkish, Chemistry, and Music 
courses. Then calculate the average grade of these courses and print "Passed" if it is 
greater than 55, and "Failed" if it is less than or equal to 55. If any of the entered 
grades are not between 0 and 100, they should not be included in the calculation of 
the average.
```

```
Kullanıcıdan Matematik, Fizik, Türkçe, Kimya ve Müzik derslerinin notları istenerek 
bu derslerin not ortalamasını alan ve 55 den büyükse "Sınıfı geçti" küçük ise "Sınıfta 
Kaldı" yazdırsın. Eğer girilen ders notları 0 ve 100 arasında değilse ortalamaya 
katılmasın. 
```
