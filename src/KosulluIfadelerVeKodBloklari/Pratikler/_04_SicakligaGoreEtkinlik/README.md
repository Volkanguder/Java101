# Patika Java101
```
Here's the translation:

A program that suggests activities based on the temperature information received 
from the user. Activity Recommendation Conditions:
If the temperature is below 5, it suggests "Skiing".
If the temperature is between 5 and 15, it suggests "Going to the cinema".
If the temperature is between 15 and 25, it suggests "Having a picnic".
If the temperature is above 25, it suggests "Swimming".
```
