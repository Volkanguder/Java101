```
We are creating a program in Java to determine whether a number is a 
"Palindrome Number."

What is a Palindrome Number?
A palindrome number is a number that reads the same from both ends.

Example: 1, 4, 8, 99, 101, 363, 4004, 9889....

This program uses a method called isPalindrome to check whether a number 
entered by the user or a predefined number is palindrome. The isPalindrome 
method calculates the reverse of the given number and compares it with the 
original number to determine if it is a palindrome.

In this example, the variable number is assigned the value 9889. The isPalindrome 
method calculates the reverse of this number and compares it with the original 
number. As a result, since 9889 is a palindrome number, the program outputs 
"9889 is a palindrome number.
```

Palindrom Sayı Nedir ?

Örnek: 1, 4, 8, 99, 101, 363, 4004, 9889....

Bu program, isPalindrome adında bir metot kullanarak kullanıcıdan alınan  veya

isPalindrome metodu, alınan sayının tersini hesaplar ve bu ters sayı ile orijinal 
sayıyı karşılaştırarak palindromik olup olmadığını belirler.

Bu örnekte, number değişkenine 9889 değeri verilmiştir. isPalindrome metodu bu 
sayının tersini hesaplar ve orijinal sayıyla karşılaştırır. Sonuç olarak, 9889 
sayısı palindromik bir sayı olduğu için program "9889 bir palindromik sayıdır." 
şeklinde bir çıktı verir.
