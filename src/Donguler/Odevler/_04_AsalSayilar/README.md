# Patika Java101
```
A program in Java that prints prime numbers between 1 and 100.

In this program, two loops are used to find prime numbers between 1 and 100. The outer 
loop (i) checks numbers from 2 to 100. The inner loop (j) tries all values from 2 to 
the number to determine if a number is prime. If a number is divisible by j, the 
prime variable is marked as false, and the inner loop is exited because a number 
cannot have a divisor greater than itself. If the prime variable is still true after 
exiting the inner loop, the number is prime and printed on the screen.
```

```
Java ile 1 - 100 arasındaki asal sayıları ekrana yazdıran program.

Bu programda, 1 ile 100 arasındaki asal sayıları bulmak için iki döngü kullanılmaktadır. 
Dış döngü (i), 2'den 100'e kadar olan sayıları kontrol eder. İç döngü (j), her bir 
sayının asal olup olmadığını kontrol etmek için 2'den sayıya kadar olan tüm değerleri 
dener. Eğer bir sayı j'ye tam bölünürse, asal değişkeni false olarak işaretlenir ve iç 
döngüden çıkılır, çünkü bir sayının böleni olarak kendisinden büyük bir sayı olamaz. 
Eğer iç döngüden çıkılmadan asal değişkeni hala true ise, sayı asaldır ve ekrana 
yazdırılır.
```
