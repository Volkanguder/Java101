# Patika Java101
```
Write a program in Java that determines whether a number entered from the keyboard 
is a perfect number or not, and displays the corresponding messages "It is a perfect 
number" or "It is not a perfect number" on the screen.

What is a Perfect Number?
A number is called a perfect number if the sum of its proper positive divisors 
(excluding the number itself) is equal to the number itself.
```
