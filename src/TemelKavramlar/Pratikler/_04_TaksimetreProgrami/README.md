# Patika Java101
```
Taximeter Program
Write a Java program that calculates the taximeter fare based on the distance traveled (in kilometers) and displays it on the screen.

The taximeter charges 2.20 TL per kilometer.
The minimum fare is 20 TL. If the fare is below 20 TL, 20 TL will be charged.
The starting fee of the taximeter is 10 TL
```

```
Taksimetre Programı
Java ile gidilen mesafeye (KM) göre taksimetre tutarını ekrana yazdıran programı yazın.

Taksimetre KM başına 2.20 TL tutmaktadır.
Minimum ödenecek tutar 20 TL'dir. 20 TL altında ki ücretlerde yine 20 TL alınacaktır.
Taksimetre açılış ücreti 10 TL'dir.
```
