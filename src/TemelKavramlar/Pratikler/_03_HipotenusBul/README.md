```
Program to Find the Hypotenuse of a Right Triangle
Write a program in Java that takes the length of the legs of a right triangle from the user and calculates the hypotenuse.

Assignment
Write a program that takes the length of three sides of a triangle from the user and calculates its area.

Formula
Semiperimeter of the triangle = 2u

u = (a+b+c) / 2

Area * Area = u * (u - a) * (u - b) * (u - c)
```
