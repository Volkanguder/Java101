# Patika Java101 
```
Program for calculating the average grade:
Write a Java program that takes the exam scores of Mathematics, Physics, Chemistry, Turkish, History, and Music from the user, calculates their average, and prints it on the screen.

Task:
Within the same program, use conditional statements to print "Passed" on the screen if the user's average is greater than 60, and "Failed" if it is less than 60.

Note: Do not use If and Else statements.
```
