# Patika Java101
```
Program for Calculating TAX:
Write a Java program that calculates and prints on the screen the TAX-inclusive price and TAX amount of the money value taken from the user.

(Note: Assume the TAX rate to be 18%)

Price without TAX = 10;

Price with TAX = 11.8;

TAX amount = 1.8;

Task:
Write a program that calculates the TAX amount based on the following conditions: If the amount entered is between 0 and 1000 TL, the TAX rate is 18%, and if the amount is greater than 1000 TL, the TAX rate is 8%.

(Note: TL stands for Turkish Lira)
```

(Not : KDV tutarını 18% olarak alın)

KDV'siz Fiyat = 10;

Ödev
