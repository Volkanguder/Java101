```
Body Mass Index Calculation
Take the height and weight values from the user and assign them to a variable in Java. Calculate the user's "Body Mass Index" value according to the following formula and print it to the screen.

Formula
Weight (kg) / Height(m) * Height(m)

Output
Please enter your height (in meters): 1.72
Please enter your weight: 105
Your Body Mass Index: 35.49215792320173"
```
