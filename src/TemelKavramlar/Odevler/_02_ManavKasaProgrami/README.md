# Patika Java101
```
Greengrocer Cashier Program
Write a program in Java that calculates and prints the total price of the products purchased by users at the grocery store based on their kilogram values.

Fruits and KG Prices:

Pear: 2.14 TL
Apple: 3.67 TL
Tomato: 1.11 TL
Banana: 0.95 TL
Eggplant: 5.00 TL

Sample Output:

How many kilograms of pear? : 0
How many kilograms of apple? : 1
How many kilograms of tomato? : 1
How many kilograms of banana? : 2
How many kilograms of eggplant? : 3
Total Price: 21.68 TL
```
